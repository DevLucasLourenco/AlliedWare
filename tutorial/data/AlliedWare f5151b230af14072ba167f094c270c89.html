Tutorial

Tutorial de utilização do Allied.

Exportar

➡️ Ao utilizar a opção de Exportar, será deparado com 3 opções, sendo responsável por exportar em XLSX, JSON e criar um servidor Streamlit, respectivamente.

Log

➡️ Ao utilizar a opção de Log, você será capaz de analisar 2 opções, o Log Geral, onde é aglomerada toda informação desde sua primeira instanciação, e Log da Instância, cujo é responsável por todo log gerado na instância atual.

Alocador

➡️ Ao utilizar a opção de Alocadores, você será capaz de realizar 2 ações:

Informar um novo JSON de apontamento, ou visualizar o apontamento atual.

Diretório de Orientação

➡️ No ambiente de orientação, este é o local (Diretório) que informará onde será analisado os códigos* .
  • O Data Counter presente abaixo da área de inserção e exibição do Diretório de Orientação apresenta a contagem de códigos* encontrados

Códigos* = DIF - CC - CP - HE

Execuções

➡️ Cada um dos botões apresentados será responsável por executar as tarefas de DIF, CC, CP e HE.

Inicialmente, estes estão bloqueados devido à não apresentação de um

Diretório de Orientação. Após apresentar, eles estarão liberados para utilização.

Executor de Scripts

➡️ O executor de Scripts estará sempre disponível para uso. Nele, todos os scripts que forem desenvolvidos estarão aptos para sua utilização.

Ao selecionar um script para ser utilizado, ao posicionar o mouse por cima deste, será apresentado um breve resumo de sua Task.

Códigos para Apontamentos

Descrição Código Atividade Exemplo de Uso
Documento de Informação de Funcionário DIF - Alocar os arquivos demarcados como DIF para a respectiva pasta do colaborador. DIF - RG - LUCAS LOURENÇO.pdf
Contracheque CC - Alocar os arquivos demarcados como CC para o respectivo mês. CC - LUCAS LOURENÇO 08.24
Cartão de Ponto CP - Alocar os arquivos demarcados como para o respectivo período. CP - LUCAS LOURENÇO - 08.2024
Solicitação de HE Solicitação de HE - Alocar os arquivos demarcados como para o respectivo período e subpasta referente. SOLICITAÇÃO DE HE - ADM 18.08.2024
